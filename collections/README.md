# Manual Collections (.http)

Arquivos para teste manual do backend usando a extensão **HTTP Client** do VSCode.

## Como usar

1. Suba o backend em `http://localhost:3001`.
2. Abra `collections/backend.http`.
3. Clique em **Send Request** acima de cada bloco.

## Arquivos

- `backend.http`: health check, listagem, busca, ordenação e detalhe de pokemon.
